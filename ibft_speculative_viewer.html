IBFT Speculative Execution Timeline Viewer

Speculative execution model: validators begin executing as soon as the first bytes of the proposal arrive (streaming decode), not after the full block body is received. We model rx-start as p50/p95 based on Δ, queueing, and sender scheduling tail.

Parameters

Computed cw: ms rx-start p50: ms rx-start p95: ms
Green bar = cw. Validator p95 exec finishing after cw end implies quorum misses deadline (round-change risk).

Timeline

Validators start executing on first bytes (streaming): rx-start = execEnd + Δ + queue + q·pp_eff. p50 is solid, p95 is dashed outline.
